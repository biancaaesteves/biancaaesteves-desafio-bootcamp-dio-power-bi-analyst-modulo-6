# 📊 Relatórios Dinâmicos com Parâmetros no Power BI

## 🎯 Objetivo do Projeto

Projeto desenvolvido durante a **Formação Power BI Analyst (DIO)** no módulo **Gerenciamento de Workspaces e Datasets com Power BI**.

O desafio teve como objetivo criar uma página analítica utilizando **Parâmetros de Campo (Field Parameters)** para desenvolver relatórios dinâmicos e interativos, permitindo que o usuário altere dimensões e métricas sem necessidade de recriação dos visuais.

Além da implementação técnica, o projeto também teve foco em **storytelling**, **experiência do usuário (UX)** e construção de uma visão analítica voltada para apresentação ao cliente.

## 📚 Estrutura Analítica

Foi criada uma página dedicada chamada **Relatório Dinâmico**, permitindo alterar visualizações e métricas de forma interativa.

O objetivo foi construir uma experiência onde o usuário pudesse explorar os dados sob diferentes perspectivas sem necessidade de múltiplos relatórios estáticos.

---

## 📈 Primeira Visão — Parâmetros por Categoria

O primeiro visual utiliza parâmetros baseados em dimensões analíticas.

Categorias implementadas:

- Country
- Month Name
- Product
- Semesters
- Segment

O usuário pode alternar dinamicamente entre essas dimensões e modificar o contexto da análise.

### Exemplo de uso:

**TOP 3 Produtos por País** → análise por **Segmento** → análise por **Produto** → análise temporal.

Essa abordagem reduz retrabalho e aumenta a flexibilidade do relatório.

---

## 📊 Segunda Visão — Parâmetros por Valores

Foi implementado um segundo grupo de parâmetros voltado para métricas.

Medidas utilizadas:

- Sales
- AVG Sales
- Profit
- Profit Previous Month

O usuário pode alterar dinamicamente os indicadores exibidos no visual.

Também foi aplicada:

✅ Formatação condicional para valores negativos

Melhorando a leitura e interpretação dos resultados.
